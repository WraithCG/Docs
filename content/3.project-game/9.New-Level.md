# How to Play in a New Level

To test the systems in a fresh environment, follow these steps:

1.  **Create Level:** Go to **File > New Level**.
2.  **Select Type:** Choose **Basic** (or Open World/Empty depending on preference).
3.  **GameMode Setup:**
    * Open the **World Settings** tab.
    * Locate the **GameMode Override** section.
    * Set the GameMode to **GM_Project**.
4.  **Play:** Press Play. The GameMode will automatically handle spawning the character, initializing the camera, and loading the default UI.
