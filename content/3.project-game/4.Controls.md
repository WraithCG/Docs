# Controls & Gameplay

The project comes with a standard third-person control scheme set up for keyboard and mouse interaction.

## Default Keybindings

| Action | Key / Input | Description |
| :--- | :--- | :--- |
| **Movement** | **W, A, S, D** | Directional movement (Forward, Left, Backward, Right) |
| **Jump** | **Spacebar** | Character jumps |
| **Crouch** | **C** | Toggles crouch state |
| **Aim** | **Right Mouse Button** | Enters aiming camera mode |
| **Pause** | **TAB** | Opens the Pause Menu (Gameplay HUD displays this prompt) |

*Note: Input mappings can be modified in the Input Action settings found in the project content browser.*
